# 温度数据采集模块

## 模块概述

温度数据采集模块 (`temp_data_collector.py`) 是专为3D打印机加热系统热参数辨识而设计的数据采集工具。该模块提供两种实验方法用于全面表征加热系统的热学特性：

### 实验方法

1. **稳态阶梯响应实验 (Steady-State Staircase Characterization)**
   - **目的**：分离并量化系统的静态热损耗参数
   - **辨识参数**：
     - 线性散热系数 $K_{lin}$（包含对流和喉管传导）
     - 非线性辐射系数 $K_{rad}$（即 $\varepsilon \sigma A$）
   - **原理**：在热平衡状态下，温度变化率为零（$dT/dt = 0$），输入功率完全等于散热功率

2. **高频动态激励实验 (PRBS Dynamic Excitation)**
   - **目的**：辨识系统的动态惯性参数
   - **辨识参数**：
     - 各节点热容 $C_h, C_b, C_s$
     - 节点间接触热阻 $R_{hb}, R_{bs}$
   - **原理**：通过伪随机二进制序列激发系统的频率响应特性

## 热学模型

### 热损耗模型

总热损耗可建模为线性项（对流+传导）与非线性项（辐射）之和：

$$P_{loss} = K_{lin} \cdot (T - T_{amb}) + K_{rad} \cdot (T^4 - T_{amb}^4)$$

其中：
- $P_{loss}$：热损耗功率 (W)
- $K_{lin}$：线性散热系数 (W/K)
- $K_{rad}$：辐射系数 (W/K⁴)，等于 $\varepsilon \sigma A$
- $T$：绝对温度 (K)
- $T_{amb}$：环境温度 (K)

### 多节点热网络模型

```
[加热芯] --R_hb--> [加热块] --R_bs--> [传感器]
    |                  |                  |
  C_h                C_b                C_s
    |                  |                  |
 热损耗             热损耗             热损耗
```

参数说明：
- $C_h, C_b, C_s$：加热芯、加热块、传感器的热容 (J/K)
- $R_{hb}, R_{bs}$：节点间的接触热阻 (K/W)

## 安装配置

### 安装步骤

**步骤 1：复制模块文件**

将 `temp_data_collector.py` 模块文件复制到 Klipper 的 extras 目录：

```bash
cp temp_data_collector.py ~/klipper/klippy/extras/
```

**步骤 2：创建数据存储目录**

```bash
mkdir -p ~/printer_data/temp_data
```

**步骤 3：修改配置文件**

在 `printer.cfg` 中添加模块配置（见下方配置示例）。

**步骤 4：重启 Klipper 服务**

添加新的 Python 模块后，需要重启 Klipper 服务以加载模块：

```bash
sudo systemctl restart klipper
```

或者使用 Klipper 常用的重启命令：

```bash
# 方法一：通过 systemctl 重启
sudo systemctl restart klipper

# 方法二：如果使用 Moonraker 管理
sudo systemctl restart klipper-moonraker

# 方法三：通过 firmware_restart 重启（在控制台执行）
FIRMWARE_RESTART
```

**步骤 5：验证安装**

重启后，在 Klipper 控制台执行以下命令验证模块是否加载成功：

```gcode
TEMP_DATA_STATUS
```

如果模块加载成功，将显示当前采集器状态信息。

### 配置示例

在 `printer.cfg` 中添加以下配置：

```ini
[temp_data_collector]
# 数据存储目录（支持 ~ 展开为用户主目录）
data_dir: ~/printer_data/temp_data
# 默认采样频率 (Hz)
sample_rate: 10.0
# 加热器最大功率 (W)，用于PWM转功率计算
max_heater_power: 60.0
```

### 配置参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `data_dir` | `~/printer_data/temp_data` | 数据文件存储目录 |
| `sample_rate` | `10.0` | 默认采样频率 (Hz) |
| `max_heater_power` | `60.0` | 加热器额定功率 (W) |

### STEADY_STATE_CALIBRATE - 稳态阶梯响应实验

运行稳态阶梯响应校准实验，测量不同温度点下的平衡功率。

**原理说明：**

在热平衡状态下，$dT/dt = 0$，此时：
$$P_{in} = P_{loss}(T) = K_{lin} \cdot (T - T_{amb}) + K_{rad} \cdot (T^4 - T_{amb}^4)$$

通过测量多个温度点的平衡功率，可以分离出线性散热系数和辐射系数。

**语法：**
```
STEADY_STATE_CALIBRATE [HEATER=<加热器>] [TEMP_POINTS=<温度点>] 
                       [TOLERANCE=<容差>] [DURATION=<持续时间>] 
                       [FILENAME=<文件名>]
                       [COOLING_ENABLED=<0|1>]
                       [COOLING_MODE=<duration|target_temp>]
                       [COOLING_DURATION=<秒>]
                       [COOLING_TARGET_TEMP=<温度>]
                       [F=<挤出速度>] [EXTRUDE_DURATION=<挤出时长>]
```

**参数：**
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `HEATER` | `extruder` | 加热器名称 |
| `TEMP_POINTS` | `50,100,150,200,250,300` | 温度设定点序列 (°C) |
| `TOLERANCE` | `1` | 稳定性判断容差 (°C) |
| `DURATION` | `180` | 稳定持续时间要求 (秒) |
| `FILENAME` | 自动生成 | 输出文件名 |
| `COOLING_ENABLED` | `1` | 是否启用冷却数据采集（0=禁用，1=启用） |
| `COOLING_MODE` | `target_temp` | 冷却模式：`duration`（固定时长）或 `target_temp`（目标温度） |
| `COOLING_DURATION` | `600` | 冷却持续时间（秒），仅在 `duration` 模式下有效 |
| `COOLING_TARGET_TEMP` | `30` | 冷却目标温度 (°C)，仅在 `target_temp` 模式下有效 |
| `F` | `0` | 挤出速度 (mm/min)，0=不挤出，>0则启用挤出 |
| `EXTRUDE_DURATION` | `0` | 挤出持续时间 (秒)，0=挤出到稳态采集结束 |

**冷却数据采集说明：**

完成所有稳态温度点测量后，系统会自动关闭加热器并采集自然冷却数据。冷却数据对于分析热损耗模型非常重要，因为冷却过程只包含纯散热，没有加热功率干扰，可以更准确地辨识散热参数。

两种冷却模式：
- **固定时长模式 (`duration`)**：采集指定时长的冷却数据，适合需要固定时间窗口的分析
- **目标温度模式 (`target_temp`)**：持续采集直到温度降至目标值，适合需要完整冷却曲线的分析

**挤出功能说明：**

在稳态数据采集阶段，可选择启用耗材挤出功能，模拟实际打印时的热负载条件。该功能为可选项，通过 `F` 参数控制：

- `F=0`（默认）：不挤出，仅采集静态稳态数据
- `F>0`：启用挤出，在温度达到稳态后自动开始挤出耗材

**挤出时序：**
1. 温度达到目标值并确认稳定（5秒窗口验证）
2. 进入稳态数据采集阶段
3. 稳态采集开始后等待15秒（硬编码 `STABILITY_BEFORE_EXTRUDE`）
4. 开始挤出耗材，持续至稳态采集结束或指定时长

**注意事项：**
- 挤出使用相对挤出模式（M83），实验结束后自动恢复原有挤出模式
- 每个温度点都会执行挤出（如果启用）
- 确保挤出机已正确配置和预热

**示例：**
```gcode
; 标准校准（默认冷却至30°C）
STEADY_STATE_CALIBRATE HEATER=extruder TEMP_POINTS=50,100,150,200,250,300

; 高温校准（冷却至40°C）
STEADY_STATE_CALIBRATE HEATER=extruder TEMP_POINTS=50,100,150,200,250,300,350,400,450 COOLING_TARGET_TEMP=40

; 固定时长冷却模式（冷却600秒）
STEADY_STATE_CALIBRATE HEATER=extruder COOLING_MODE=duration COOLING_DURATION=600

; 禁用冷却数据采集
STEADY_STATE_CALIBRATE HEATER=extruder COOLING_ENABLED=0

; 启用挤出（300mm/min，挤出到采集结束）
STEADY_STATE_CALIBRATE HEATER=extruder F=300

; 启用挤出（300mm/min，挤出60秒）
STEADY_STATE_CALIBRATE HEATER=extruder F=300 EXTRUDE_DURATION=60
```

**输出文件：**
- `<filename>`：原始时序数据（包含 `phase` 列标记加热/冷却阶段）
- `<filename>_results.csv`：各温度点的汇总结果
- `<filename>_cooling.csv`：冷却阶段统计信息

### PRBS_CALIBRATE - PRBS动态激励实验

运行伪随机二进制序列动态激励实验，辨识系统的动态热参数。

**原理说明：**

PRBS信号包含丰富的频率成分：
- **短脉冲（0.2-1秒）**：激发小热容节点（加热芯）的快速响应
- **长脉冲（5-20秒）**：让热量有足够时间穿透接触热阻，反映整体热容和传导滞后

该信号满足系统辨识理论中的"持续激励条件"，保证辨识结果收敛且唯一。

**语法：**
```
PRBS_CALIBRATE [HEATER=<加热器>] [DURATION=<总时长>] 
               [MIN_PULSE=<最小脉宽>] [MAX_PULSE=<最大脉宽>]
               [POWER_LEVELS=<功率电平>] [SAMPLE_RATE=<采样率>]
               [MAX_TEMP=<最高温度>] [FILENAME=<文件名>]
```

**参数：**
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `HEATER` | `extruder` | 加热器名称 |
| `DURATION` | `300` | 实验总时长 (秒) |
| `MIN_PULSE` | `0.2` | 最小脉冲宽度 (秒) |
| `MAX_PULSE` | `10.0` | 最大脉冲宽度 (秒) |
| `POWER_LEVELS` | `0.0,1.0` | 功率电平序列 (0.0-1.0) |
| `SAMPLE_RATE` | `20.0` | 采样频率 (Hz) |
| `MAX_TEMP` | `300.0` | 最高温度限制 (°C)，超过此温度将暂停加热 |
| `FILENAME` | 自动生成 | 输出文件名 |

**安全机制：**

该命令内置温度保护机制，确保实验安全进行：

1. **温度超限保护**：当监测温度 ≥ `MAX_TEMP` 时，立即关闭加热器
2. **自动恢复**：当温度降至 `MAX_TEMP - 100°C` 时，自动恢复实验
3. **时间统计**：暂停期间的时间不计入有效实验时长，实验结束后会报告：
   - 实际运行时长
   - 暂停时长
   - 有效实验时长

**示例：**
```gcode
; 标准PRBS实验（默认最高温度300°C）
PRBS_CALIBRATE HEATER=extruder DURATION=300 MIN_PULSE=0.2 MAX_PULSE=10

; 高温打印校准（设置最高温度280°C）
PRBS_CALIBRATE HEATER=extruder DURATION=600 MIN_PULSE=0.2 MAX_PULSE=15 POWER_LEVELS=0.0,0.5,1.0 MAX_TEMP=280

; 热床PRBS实验（较低温度限制）
PRBS_CALIBRATE HEATER=heater_bed DURATION=300 MAX_TEMP=120
```

### THERMAL_ID_CALIBRATE - 综合热参数辨识实验

运行10分钟综合热参数辨识实验，一次性完成所有热参数辨识。

**原理说明：**

该实验是一个完整的5阶段热参数辨识流程，总时长约10分钟：

```
时间轴（秒）：
├── 0-50 ────┼── 50-160 ────┼── 160-260 ───┼── 260-380 ───┼── 380-600 ──┤
│ 阶段1      │   阶段2      │   阶段3      │   阶段4      │   阶段5     │
│ 初始阶跃   │  200°C稳态   │  300°C稳态   │  350°C稳态   │  安全PRBS  │
│ (50秒)     │  (110秒)     │  (100秒)     │  (120秒)     │  (220秒)    │
│            │              │              │              │             │
│ 辨识θ_8    │ 辨识散热参数 │ 辨识散热参数 │ 辨识散热参数 │ 辨识动态参数│
│            │              │              │ (含辐射项)   │             │
```

**各阶段辨识目标：**

| 阶段 | 控制方式 | 辨识目标 |
|------|----------|----------|
| 阶段1：初始阶跃 | 开环满功率 | θ_8（初始升温斜率）、整体热容 |
| 阶段2：200°C稳态 | PID闭环 | 中温区散热功率 |
| 阶段3：300°C稳态 | PID闭环 | 高温区散热功率 |
| 阶段4：350°C稳态 | PID闭环 | 辐射散热参数（高温非线性） |
| 阶段5：安全PRBS | 开环带保护 | 动态热参数（热容、热阻） |

**语法：**
```
THERMAL_ID_CALIBRATE [HEATER=<加热器>] [SAMPLE_RATE=<采样率>]
                     [MAX_TEMP=<最高温度>] [FILENAME=<文件名>]
                     [STEP1_DURATION=<秒>]
                     [STEP2_TEMP=<温度>] [STEP2_DURATION=<秒>]
                     [STEP3_TEMP=<温度>] [STEP3_DURATION=<秒>]
                     [STEP4_TEMP=<温度>] [STEP4_DURATION=<秒>]
                     [STEP5_DURATION=<秒>]
                     [PRBS_LOW_POWER=<比例>] [PRBS_HIGH_POWER=<比例>]
```

**参数：**
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `HEATER` | `extruder` | 加热器名称 |
| `SAMPLE_RATE` | `20.0` | 采样频率 (Hz) |
| `MAX_TEMP` | `400.0` | 最高温度限制 (°C) |
| `FILENAME` | 自动生成 | 输出文件名 |
| `STEP1_DURATION` | `50` | 阶段1持续时间（秒） |
| `STEP2_TEMP` | `200` | 阶段2目标温度 (°C) |
| `STEP2_DURATION` | `110` | 阶段2持续时间（秒） |
| `STEP3_TEMP` | `300` | 阶段3目标温度 (°C) |
| `STEP3_DURATION` | `100` | 阶段3持续时间（秒） |
| `STEP4_TEMP` | `350` | 阶段4目标温度 (°C) |
| `STEP4_DURATION` | `120` | 阶段4持续时间（秒） |
| `STEP5_DURATION` | `220` | 阶段5持续时间（秒） |
| `PRBS_LOW_POWER` | `0.2` | PRBS低功率电平 (20%) |
| `PRBS_HIGH_POWER` | `0.6` | PRBS高功率电平 (60%) |

**安全机制：**

阶段5（PRBS）内置多层温度保护：
1. **温度上限保护**：超过 `MAX_TEMP` 时强制关闭加热器
2. **温度下限保护**：低于 `MAX_TEMP - 100°C` 时自动提升功率
3. **动态功率调整**：接近温度上限时自动切换到低功率
4. **时间统计**：保护暂停时间不计入有效实验时长

**示例：**
```gcode
; 标准综合校准（默认参数，约10分钟）
THERMAL_ID_CALIBRATE HEATER=extruder

; 自定义温度配置
THERMAL_ID_CALIBRATE HEATER=extruder STEP2_TEMP=180 STEP3_TEMP=260 STEP4_TEMP=320 MAX_TEMP=380

; 热床校准（较低温度范围）
THERMAL_ID_CALIBRATE HEATER=heater_bed STEP2_TEMP=60 STEP3_TEMP=80 STEP4_TEMP=100 MAX_TEMP=120

; 自定义PRBS功率范围
THERMAL_ID_CALIBRATE HEATER=extruder PRBS_LOW_POWER=0.3 PRBS_HIGH_POWER=0.7
```

**输出文件：**
- `<filename>`：原始时序数据（包含 `phase` 列标记各阶段）
- `<filename>_results.csv`：各阶段汇总结果

**结果文件格式：**

| 列名 | 说明 |
|------|------|
| `phase` | 阶段标识（step1-step5） |
| `parameter` | 参数名称 |
| `value` | 参数值 |
| `unit` | 单位 |

**实验结束后输出摘要示例：**
```
实验结果摘要:
----------------------------------------
θ_8估计 (初始升温斜率): 2.345 °C/s

稳态散热功率:
  200°C: P = 45.23W (T_avg = 199.8°C)
  300°C: P = 78.56W (T_avg = 299.5°C)
  350°C: P = 102.34W (T_avg = 349.2°C)

PRBS动态激励: 35个脉冲, 有效时长 215.3s
----------------------------------------
```

## 数据输出格式

### 时序数据文件 (CSV)

| 列名 | 数据类型 | 说明 |
|------|----------|------|
| `time` | float | 相对时间（秒，从实验开始计时） |
| `temperature` | float | 当前温度 (°C) |
| `pwm` | float | PWM占空比 (0.0-1.0) |
| `target` | float | 目标温度 (°C) |
| `power_watts` | float | 计算功率 (W) |
| `experiment` | string | 实验名称 |
| `phase` | string | 当前阶段（`heating` 或 `cooling`） |

### 稳态结果文件 (CSV)

| 列名 | 数据类型 | 说明 |

## 注意事项与安全警告

### 安全警告

1. **PRBS实验使用开环控制**：该实验直接设置PWM占空比，绕过PID控制器。现已内置温度保护机制：
   - **MAX_TEMP 参数**：设置最高温度限制（默认 300°C）
   - **自动暂停**：温度超限时自动关闭加热器
   - **自动恢复**：温度降至安全阈值后自动继续实验
   - 建议根据加热器类型设置合理的 `MAX_TEMP` 值

2. **温度范围**：确保所有温度设定点在加热器的安全工作范围内。

3. **实验监控**：虽然有自动保护机制，仍建议实验期间有人监控，以防意外情况。

### 实验建议

1. **稳态实验前**：
   - 关闭所有主动散热风扇（模型冷却风扇、喉管散热风扇）
   - 保持环境温度恒定（约25°C）
   - 排除外界气流干扰

2. **采样率选择**：
   - 稳态实验：5-10 Hz 即可
   - PRBS实验：建议 15-20 Hz，以捕捉快速瞬变

3. **数据存储**：
   - 20 Hz 采样率下，10分钟实验约产生 12,000 个样本
   - 确保有足够的磁盘空间

## 高级分析建议

### 冷却数据分析

冷却数据是分析热损耗模型的宝贵资源，因为冷却过程只包含纯散热，没有加热功率干扰。通过分析冷却曲线可以：

1. **验证散热模型**：冷却过程中的温度变化率直接反映散热特性
2. **估计时间常数**：从指数衰减曲线中提取热时间常数
3. **分离散热分量**：区分线性散热（对流/传导）和非线性散热（辐射）

**Python分析示例：**

```python
import pandas as pd
import numpy as np
from scipy.optimize import curve_fit

# 加载数据
data = pd.read_csv('steady_state_xxx.csv')

# 分离冷却数据
cooling_data = data[data['phase'] == 'cooling']
time = cooling_data['time'].values - cooling_data['time'].values[0]
temp = cooling_data['temperature'].values

# 拟合指数冷却模型
def cooling_model(t, T0, T_amb, tau):
    """指数冷却模型: T(t) = T_amb + (T0 - T_amb) * exp(-t/tau)"""
    return T_amb + (T0 - T_amb) * np.exp(-t / tau)

# 初始猜测
T0_guess = temp[0]
T_amb_guess = 25.0
tau_guess = 100.0

# 拟合
popt, pcov = curve_fit(cooling_model, time, temp, p0=[T0_guess, T_amb_guess, tau_guess])
T0_fit, T_amb_fit, tau_fit = popt

print(f"初始温度: {T0_fit:.1f}°C")
print(f"环境温度: {T_amb_fit:.1f}°C")
print(f"时间常数: {tau_fit:.1f}s")
```
